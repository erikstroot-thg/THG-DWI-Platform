## Gebruik in Code

### Pad in React app:
```javascript
// In werkinstructies.js — per stap:
{
  tekst: 'Druk op de groene knop in de gele behuizing',
  afbeeldingen: ['/images/dwi/DWI-BOR-001/stap-01.jpg'],
  bijschrift: ['Groene = AAN | Rode = UIT']
}
```

### In DetailPagina.jsx — render:
```jsx
{stap.afbeeldingen?.map((src, i) => (
  <figure key={i} className="my-4">
    <img src={src} alt={stap.bijschrift?.[i] || `Stap ${nr}`}
         className="w-full rounded-lg shadow" />
    {stap.bijschrift?.[i] && (
      <figcaption className="text-sm text-gray-600 mt-1">
        {stap.bijschrift[i]}
      </figcaption>
    )}
  </figure>
))}
```
